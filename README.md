# PI_MathQuest

**Edukacyjna gra 3D wspierająca naukę metod numerycznych.**
Projekt zrealizowany jako praca inżynierska na Politechnice Gdańskiej (2025).

---

## O grze
Wcielasz się w inżyniera w bazie na Marsie. Twoim zadaniem jest naprawa uszkodzonych systemów poprzez rozwiązywanie zadań matematycznych na terminalach komputerowych.

**Zakres materiału:**
* Równania nieliniowe (np. metoda bisekcji, Newtona)
* Układy równań liniowych (np. eliminacja Gaussa)
* Interpolacja i aproksymacja
* Całkowanie numeryczne

## Jak pobrać i zagrać?
Gra nie wymaga instalacji.

1. Przejdź do sekcji **[Releases](https://github.com/kaspeo/PI_MathQuest/releases)** (po prawej stronie).
2. Pobierz najnowszy plik `.zip`.
3. Rozpakuj archiwum w dowolnym folderze.
4. Uruchom plik **`MathQuest.exe`**.

> **Uwaga:** Jeśli system Windows wyświetli komunikat o zabezpieczeniach , kliknij **"Więcej informacji"** a następnie **"Uruchom mimo to"**.
