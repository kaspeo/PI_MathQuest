# PI_MathQuest
MathQuest – Praca inżynierska (PG). Edukacyjna gra 3D w silniku Godot, wspierająca naukę metod numerycznych. Gracz naprawia bazę na Marsie, rozwiązując zadania z algebry liniowej, interpolacji i całkowania (m.in. metoda Newtona, Gaussa, Monte Carlo) . Projekt zawiera wizualizację algorytmów w czasie rzeczywistym.
